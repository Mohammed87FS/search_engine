# Build Scripts

This directory contains various build scripts used during development:

- `build_gui_fixed.bat` - Fixed version of GUI build script
- `build_gui_simple.bat` - Simplified GUI build script
- `build_gui_verbose.bat` - Verbose GUI build script for debugging
- `build_verbose.bat` - Verbose CLI build script
- `compile_all.bat` - Step-by-step compilation script
- `FINAL_BUILD.bat` - Final build script with error handling

For normal use, just use `build.bat` or `build_gui.bat` in the root directory.
