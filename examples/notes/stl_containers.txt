STL Containers

The Standard Template Library provides many useful containers:

- std::vector: Dynamic array, fast random access
- std::unordered_map: Hash table for key-value pairs
- std::string: String container with many utility functions
- std::set: Sorted unique elements

STL algorithms work with iterators to process container elements efficiently.
Use std::sort, std::find_if, and std::transform for common operations.
